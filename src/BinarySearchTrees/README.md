# Binary Search Trees

TypeScript implementation of a balanced binary search tree with traversal, search, insertion, deletion, height/depth calculation, and rebalancing helpers.

## Files

| File | Purpose |
| --- | --- |
| `TreeNode.ts` | Node model used by the tree |
| `BalancedBST.ts` | Balanced binary search tree implementation |
| `prettyPrint.ts` | Console helper for visual tree output |
| `bstDriverScript.ts` | Manual driver script for local experiments |
| `tests/balancedBST.test.ts` | Jest tests for tree behavior |

## Test command

From the repository root:

```bash
npm test -- src/BinarySearchTrees/tests/balancedBST.test.ts
```

See the root [README](../../README.md) for full setup and verification commands.
